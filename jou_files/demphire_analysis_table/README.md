# Генератор .jou файлов для демпфирования

Скрипт `generate_jou_files.py` генерирует набор .jou файлов с разными значениями параметров L, DW, R.

## Структура

Каждый сгенерированный .jou файл состоит из трех частей:

1. **template_1.jou** - геометрия и материалы (с подстановкой значений L, DW, R)
2. **template_2.jou** - динамически генерируемые таблицы с параметрами демпфирования
3. **template_3.jou** - блоки, граничные условия и параметры расчета (с подстановкой пути сохранения)

## Использование

1. Отредактируйте функцию `main()` в скрипте, задав нужные значения:
   ```python
   L_values = [400, 500, 600]  # Значения параметра L
   DW_values = [100, 150, 200]  # Значения параметра DW
   R_values = [0.1, 0.2, 0.3]  # Значения параметра R
   c_p = 3000  # Скорость волны P
   ```

2. Запустите скрипт:
   ```bash
   python generate_jou_files.py
   ```

3. Сгенерированные файлы будут сохранены в директории `generated/`

## Формулы

Для каждой таблицы создается `DW // 10 + 1` строк.

**Первая колонка (x/y координаты):**
- Таблица 1 и 3: `-(L - DW + 10 * ind_row)` (значения с минусом)
- Таблица 2: `L - DW + 10 * ind_row` (значения с плюсом)
- где `ind_row` от 0 до `DW//10`

**Вторая колонка (значения демпфирования R):**
- `R_(ind_row) = b_0 * ((L - DW + 10 * ind_row) / DW)^3`
- где `b_0 = -4 * c_p * ln(R) / 2 / DW`
